# OpenUI for OpenGoBot

Alternate UI for OpenPoGo Bot
Needs node.js to launch locally, available at http://openpogoui.nicontoso.eu/

## How to

The 'socket' plugin has to be enabled, the 'web' plugin should be deactivated.
